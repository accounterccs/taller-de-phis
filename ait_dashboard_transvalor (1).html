Manual de Extracción e Integración Automática

Para automatizar la carga de datos sin usar archivos manuales, tu equipo técnico puede utilizar las siguientes especificaciones estructurales:

1. Consulta SQL (Extracción Transaccional de SAP / BD)

SELECT 
    id_remesa AS id,
    cliente_nombre AS cliente,
    numero_comprobante AS comprobante,
    monto_solicitado AS amount_expected,
    monto_contado AS amount_real,
    moneda_tipo AS currency,
    fase_actual AS status,
    fecha_creacion AS created_at
FROM transvalor_remesas_historico
WHERE fecha_creacion >= CURRENT_DATE - 15;

2. Integración Python (Posteo de Lote de Datos)

import requests

api_url = "https://tu-endpoint.transvalor.com/api/v1/sync"
payload = {
    "auth_token": "SUAREZ_SECURE_TOKEN_2026",
    "remittances": [
        {"id": "REM-2026-99", "client_id": "C01", "comprobante": "COMP-1011", "amount": 2500, "currency": "Dólar", "status": "Solicitud", "channel": "ATM"}
    ]
}
response = requests.post(api_url, json=payload)
